## vscode插件

#### Paste Image
> 写markdown文档可快速保存图片并展示。
```
# 优化图片目录，修改插件配置：
base Path: ${currentFileDir}/.img
insert Pattern: ${imageSyntaxPrefix}.img/${imageFilePath}${imageSyntaxSuffix}
Path: ${currentFileDir}/.img
```
